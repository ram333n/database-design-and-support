Предметна область: Готель.

1) Є номери готелю, про які відомо наступне: номер, площа, к-ть одномісних ліжок, к-ть двомісних ліжок. 
Клієнт може забронювати номер, при цьому зберігається інформація про замовлення: ціна, дата заїзду, дата виїзду.
Для бронювання декількох номерів потрібно створювати окреме замовлення для кожного номера окремо.

2) Для номерів готелю можна відокремити типи номерів: звичайні та президентські. Для звичайних номерів потрібно зберігати інформацію
про наявність дитячої зони. Для президентських номерів потрібно зберігати інформацію, чи може номер бути використаним для
заселення поважних гостей(наприклад, іноземних делегацій, іноземних чиновників)

3) У БД зберігається інформація про працівників, а саме: паспорт, ПІБ, посада. Також, для кожного працівника потрібно зберігати інформацію 
про те, хто його керівник.

4) Потрібно зберігати відомість про те, хто відповідальний за номер(наприклад, чистота, наявність інвентарю). Один працівник може бути відповідальним
за декілька номерів, а за один номер може відповідати декілька працівників.
